Decimal pipes y porcentajes

pipes incluidos en angular - usualmente en el common module

  • ventas_value
  • ventas_value | number
  • ventas_netas | number:'1.2-2'
  • (ventas_netas + 1500000) | number:'1.2-2'
  • {{ ventas_netas}}
  • {{ ventas_netas | number }}

  • (ventas_netas + 2000000) | currency
  • (ventas_netas + 2000000) | currency: 'CO'
  • (ventas_netas + 2000000) | currency: '': 'symbol-narrow'
  • (ventas_netas + 2000000) | currency: '': 'symbol-narrow': '1.0-4'
  • {{ (ventas_netas + 2000000) | currency }}
  • {{ (ventas_netas + 2000000) | currency: 'CO' }}
  • {{ (ventas_netas + 2000000) | currency: '': 'symbol-narrow' }}
  • {{ (ventas_netas + 2000000) | currency: '': 'symbol-narrow': '1.4-4' }}

  • {{ porcentaje | percent: '2.2-2' }}
